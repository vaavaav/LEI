Este projeto consiste num jogo de tanques em que estes usam das suas armas (canhão, laser, choque) de forma a atingir os seus adversários. Ganha o ultimo que permanecer com vidas no jogo.

Para instalar o jogo, abra o ficheiro "Setup.exe" e siga as instruções indicadas.

Projeto realizado por Rúben Cerqueira (A89593) e Pedro Peixoto (A89602) (2018li1g156) Universidade do Minho, Portugal no âmbito da Unidade Curricular "Laboratórios de Informática 1" 2018.
